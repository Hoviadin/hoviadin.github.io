клонировать репозиторий
git clone ссылка на репозиторий
Создать ветку
git checkout -b "имя ветки"
переключится на ветку
git checkout "имя ветки"
добавить изменения для коммита
git add -A
создать коммит
git commit -m "название"
отправить все в репозиторий
git push origin название ветки
обновить ветку
git pull origin имя ветки из которой будем тянуть обновления
спрятать локальные изменения
git stash -u
достать локальные изменения
git stash pop
отменить коммит с изменениями
git revert id коммита
